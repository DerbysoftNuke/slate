Change API

  1. Call Change Discover first to get changed hotel list
  2. Call AvailStatus, DailyRate, Inventory and LOSRate API to get change back.

Change Discover

  • URL - https://doorway.derbysoftsec.com/v0.1/change_discover
  • Method - POST
{
    "Header" : {
        "Version": "v0.1",
        "EchoToken": "18393849028490234",
        "Channel":"EXPEDIA",
        "IATA":"0348487"
    },

    "Hotels" : [
        {
            "Hotel": "Hotel001",
            "TimeStamp": "2304304304230"
        },
        {
            "Hotel": "Hotel002",
            "TimeStamp": "2304304304230"
        }        
    ]
}

Response for above request

{
    "Header" : {
        "Version": "v0.1",
        "EchoToken": "18393849028490234",
        "Channel":"EXPEDIA",
        "IATA":"0348487"
    },
    "HasChangeHotels" : [
        {
            "Hotel": "Hotel001",
            "TimeStamp": "2304304304230"
        },
        {
            "Hotel": "Hotel002",
            "TimeStamp": "2304304304230"
        }
    ]
}

Request

Parameter Count Type Size Description
Hotels 1.n Hotel[] Hotel and timestamp list
....Hotel 1 String 32 Hotel unique code
....Timestamp 1 Int Timestamp. Ask for change status after this time.

Response

Parameter Count Type Size Description
HasChangeHotels 0.n Hotel[] Has change hotel and timestamp list
....Hotel 1 String 32 Hotel unique code
....Timestamp 1 Int Timestamp. Ask for change status after this time.
