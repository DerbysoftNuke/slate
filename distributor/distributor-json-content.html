Hotel Search

  • URL - https://go.derbysoftsec.com/go/json/hotelsearch/yourusername_in_lower_case
  • Method - POST

The Hotel Search transaction allows distributors the ability to search hotel in the DerbySoft cache based on latitude and longitude and a specified radius around these coordinates.

Please note that a search by city is not possible due to cities and their parameters being defined differently by each Distributor and Supplier. 
{
  "Header": {
    "EchoToken": "18393849028490234",
    "Channel": "EXPEDIA",
    "IATA": "0348487",
    "Language": "en-US"
  },
  "Coordinate": {
    "Longitude": -77.1141,
    "Latitude": 38.8802,
    "Distance": 10,
    "Unit": "km"
  }
}

Response for above request

{
  "Header": {
    "EchoToken": "18393849028490234",
    "Channel": "EXPEDIA",
    "IATA": "0348487",
    "Language": "en-US"
  },
  "Hotels": [
    {
      "Code": "WI-1513",
      "Name": "The Westin Arlington Gateway",
      "Longitude": -77.1141,
      "Latitude": 38.8802,
      "Proximity": 0.0215215465,
      "Unit": "km",
      "Direction": "E"
    },
    {
      "Code": "WI-9527",
      "Name": "Stephen Chou Hotel",
      "Longitude": -77.1141,
      "Latitude": 38.8802,
      "Proximity": 100,
      "Unit": "mi",
      "Direction": "E"
    }
  ]
}

Request

Parameter Count Type Size Description
Coordinate 1
....Longitude 1 Double The specific longitude.
....Latitude 1 Double The specific latitude.
....Distance 1 Int The distance of the coordination. Maximum is 100 km or 62 miles.
....Unit 1 Enum The unit of distance. Can only be "km" or "mi".

Response

Parameter Count Type Size Description
Hotels 0.n
....Code 1 String 128 Hotel unique code.
....Name 1 String 1024 Hotel name.
....Longitude 1 Double Longitude of the hotel.
....Latitude 1 Double Latitude of the hotel.
....Proximity 1 Double The proximity of hotel.
....Unit 1 Enum The unit of Proximity. Can only be "km" or "mi". "km" is kilometer. "mi" is mile.
....Direction 0.1 Enum Please find details in the Appendix

Content Change

  • URL - https://go.derbysoftsec.com/go/json/hotelcontentchange/yourusername_in_lower_case
  • Method - POST

Distributor will initiate a request to Derbysoft about hotel context in a certain time stamp after there is no change, if there is a change, we will provide them with a list of changed hotel codes.

{
  "Header" : {
    "EchoToken": "18393849028490234",
    "Channel":"EXPEDIA",
    "IATA":"0348487",
    "Language":"en-US"
  },
  "Timestamp":"2016-01-02T10:22:36"
}

Response for above request

{
  "Header" : {
    "EchoToken": "18393849028490234",
    "Channel":"EXPEDIA",
    "IATA":"0348487",
    "Language":"en-US"
  },
  "Hotels":["HI-ATTSD","CARLSON-ABCDE"]
}

Request

Parameter Count Type Size Description
Timestamp 1 String UTC time. yyyy-MM-ddTHH:mm:ss, We return the hotel code if there are changes after this timestamp.

Response

Parameter Count Type Size Description
Hotels 0.n String[] Hotel unique code that changed since LastModifyDateTime in request.
